1.	تفاوت DNS  و DHCP  و HTTP:
•	DNSبه ترجمه نام دامنه به آدرس IP می ‌پردازد.
•	DHCPوظیفه تخصیص آدرس IP و تنظیمات شبکه به دستگاه‌ ها را بر عهده دارد.
•	HTTPپروتکلی برای انتقال صفحات و داده‌ های وب است.

2.	تحلیل ترافیک کپچر شده از سایت موبومووی :
اولین قدم پس از وارد کردن آدرس وب ‌سایت، ارسال یک درخواست DNS است. این درخواست به یک سرور DNS ارسال می ‌شود تا نام دامنه به آدرس IP ترجمه شود. بسته ‌های DNS دارای دو نوع  QueryوResponse  هستند که درخواست DNS به سرور ارسال شده و پاسخ آن که شامل آدرس IP است، دریافت می ‌شود.
پس از دریافت آدرس IP، ارتباط با سرور از طریق پروتکل TCP برقرار می ‌شود و مرورگر درخواست HTTP یا HTTPS را به سرور ارسال می‌ کند. سرور نیز پس از دریافت درخواست، پاسخ داده و محتوای وب ‌سایت را به مرورگر می ‌فرستد. این پاسخ معمولاً شامل فایل ‌های  HTML، CSS، جاوا اسکریپت و تصاویر است.
پاسخ HTTP شامل یک کد وضعیت است که نشان ‌دهنده موفقیت یا شکست درخواست است. (مانند کد 404)
